# 目的
`docs/mvp1-spec.md` に記載された仕様に基づき、AIキャラクターなりきりチャット＋口調変換の Web アプリ MVP1 を実装する。

1. `docs/mvp1-spec.md` を読み、仕様・制約・DONE 定義を把握する
2. `personas/` 内のサンプルペルソナ YAML を読み、データ構造を把握する
3. Next.js プロジェクトを初期化し、実装を開始する

# 成果物
- 種別: Next.js Web アプリケーション
- 配置先: リポジトリルート
- 入力: ペルソナ定義ファイル (personas/*.yaml) + OpenRouter API キー (.env.local)
- 出力: ブラウザで動作するチャット対話 + 口調変換アプリ
